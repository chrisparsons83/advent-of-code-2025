# advent-of-code-2025

Advent of Code 2025 solutions in TypeScript.

## Setup

```bash
npm install
```

## Running a Solution

You can run TypeScript files directly without compiling first using `tsx`:

```bash
npx tsx src/day01.ts
```

Or if you prefer to compile first, then run the JavaScript:

```bash
npm run build
node dist/day01.js
```
